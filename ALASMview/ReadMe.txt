Lister-плагин для просмотра файлов в формате ALASM
---------------------------------------------------

Автор:	Михальченков Дмитрий Александрович
E-Mail:	witchcraft2001@mail.ru


Данный плагин предназначен для отображения файлов
ALASM в текстовом виде. Поддерживаются токены версии до 5.09.

Фичи:
*  Отслеживается открытие кавычек, если при окончании строки она не
   закрыта, то добавляется закрывающая кавычка в конец строки (часто
   этот метод используется для сокращения объема текста, сам компиллер
   нормально воспринимает это, но читается такой текст не очень);

*  вместо пробела после первого токена печатается табуляция, мне
   кажется так красивее;


УСТАНОВКА
---------

1. Создать папку zasmview для плагина в папке с плагинами TC.
2. Скопировать файл zasmview.wlx в эту папку.
3. Открыть меню Конфигурация>Настройка>Плагины, нажать
   кнопку Настройка в боксе "Плагины внутреннего просмотрщика .WLX".
4. Нажать "Добавить" и выбрать файл zasmview.wlx.
5. Перезапустить TC, если ранее был использован Lister для
   просмотра (только при первом вызове подгружаются плагины).
