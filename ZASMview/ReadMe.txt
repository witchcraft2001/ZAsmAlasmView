Lister-плагин для просмотра файлов в формате ZX-ASM
---------------------------------------------------

Автор:	Михальченков Дмитрий Александрович
E-Mail:	witchcraft2001@mail.ru

Данный плагин предназначен для отображения файлов
ZX-ASM (ZX-TASM, ZAsm) в текстовом виде.
Файлы ZAsm достаточно сложно точно идентифицировать,
т.к. форматом не предусмотрены какие-либо сигнатуры, 
а сам формат представляет собой слегка модифицированный
plain text (добавлено несколько управляющих кодов,
см. файл za_format.txt). 
В данном плагине принадлежность к формату определяется 
по расширениям zas, asm, $z, $a, $C, C, !z, !a, !C.
Как попытка более точно идентифицировать формат сделана
дополнительная проверка первых 2048 байт на наличие токенов,
управляющих кодов в файле, а также отсутствие кода перевода
каретки (#0A) после кода конца строки (#0D). Однако и
эта проверка иногда лагает, поэтому если отобразится не
то, что ожидается, то нажмите кнопку "1" и файл будет
отображен средствами Lister-а.

РУЧНАЯ УСТАНОВКА
----------------

1. Создать папку zasmview для плагина в папке с плагинами TC\WLX.
2. Скопировать файл zasmview.wlx в эту папку.
3. Открыть меню Конфигурация>Настройка>Плагины, нажать
   кнопку Настройка в боксе "Плагины внутреннего просмотрщика .WLX".
4. Нажать "Добавить" и выбрать файл zasmview.wlx.
5. Перезапустить TC, если ранее был использован Lister для
   просмотра (только при первом вызове подгружаются плагины).
